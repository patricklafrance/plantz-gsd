    **Gap 3 fix — Seed count mismatch:**

    The problem: `STARTER_PLANTS` is hardcoded to 5 entries. `seedStarterPlants()` ignores the `plantCountRange` parameter from onboarding entirely — all selections produce exactly 5 plants.

    **Step 1: Update `src/features/demo/actions.ts` — Accept plantCountRange parameter:**

    Change `seedStarterPlants()` function signature to accept an optional `plantCountRange` parameter:
    ```typescript
    export async function seedStarterPlants(plantCountRange?: string)
    ```

    Add a helper at the top of the function to map range strings to target counts:
    ```typescript
    const TARGET_COUNTS: Record<string, number> = {
      "1-5 plants": 5,
      "6-15 plants": 10,
      "16-30 plants": 20,
      "30+ plants": 35,
    };
    const targetCount = TARGET_COUNTS[plantCountRange ?? "1-5 plants"] ?? 5;
    ```

    **Step 2: Update seeding logic in `seedStarterPlants()`:**

    After the existing `careProfiles` query (which fetches STARTER_PLANTS — 5 profiles), add logic to fetch additional profiles if `targetCount > careProfiles.length`:

    ```typescript
    let allProfiles = [...careProfiles];

    if (targetCount > allProfiles.length) {
      const additionalNeeded = targetCount - allProfiles.length;
      const existingNames = allProfiles.map(p => p.name);
      const additionalProfiles = await db.careProfile.findMany({
        where: { name: { notIn: existingNames } },
        take: additionalNeeded,
        orderBy: { name: "asc" },
      });
      allProfiles = [...allProfiles, ...additionalProfiles];
    }
    ```

    Then change the for loop from `for (const profile of careProfiles)` to `for (const profile of allProfiles)`.

    For nicknames: use `profile.name` as the nickname (this is what the existing code already does).

    **Step 3: Update `src/components/onboarding/onboarding-banner.tsx` — Pass range to seeder:**

    Change line 45 from:
    ```typescript
    seedStarters ? seedStarterPlants() : Promise.resolve(null),
    ```
    to:
    ```typescript
    seedStarters ? seedStarterPlants(range) : Promise.resolve(null),
    ```

    This passes the selected range (e.g., "30+ plants") to the seeder function.
